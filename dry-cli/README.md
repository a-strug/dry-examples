### dry-cli
Framework for developing CLI applications
```bash
dry-cli/tv --help
dry-cli/tv on                   # => TV is on
dry-cli/tv louder               # => Getting louder by 1
dry-cli/tv quiter 2             # => Getting quiter by 2
dry-cli/tv next                 # => Switch to the next channel
dry-cli/tv ">"                  # => Switch to the next channel
dry-cli/tv "<"                  # => Switch to the previous channel
dry-cli/tv mode --nightshift    # => Nightshift mode is on
dry-cli/tv mode --no-nightshift # => Nightshift mode is off
dry-cli/tv off                  # => TV is off
```
